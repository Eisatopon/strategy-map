🔵 1. Extremal Στοιχείο

🔵 1. Extremal Element

Από πού ξεκινάς — το πιο περιορισμένο στοιχείο

Πρόβλημα: Σε 10 κουτιά υπάρχουν μπίλιες. Δείξε ότι υπάρχει κουτί με τον ελάχιστο αριθμό μπιλιών. Τι συμβαίνει αν το αφαιρέσεις;

Το ελάχιστο κουτί έχει λιγότερες μπίλιες από όλα τα άλλα. Αν το αφαιρέσεις, το επόμενο ελάχιστο είναι μεγαλύτερο — τι σου λέει αυτό για τη συνολική δομή;

Πρόβλημα: Δίνονται 2025 διανύσματα στο επίπεδο, κανένα μηδέν. Δείξε ότι υπάρχει ζεύγος με γωνία ≤ 1°.

Θεώρησε το ζεύγος με τη μεγαλύτερη γωνία. Τι συμβαίνει αν υποθέσεις ότι όλες οι γωνίες είναι > 1°; Πόσες χωράνε στον κύκλο;

Πρόβλημα: Σύνολο \(S\subset\mathbb N\) τέτοιο ώστε για κάθε \(n\) το \(\{s\in S: s\le n\}\) έχει μέγεθος ≤ \( \log_2n \). Δείξε ότι υπάρχούν δύο στοιχεία του \(S\) με απόσταση **ακριβώς** 10.

Χτίσε τον μεγαλύτερο δυνατό \(S\) που αποφεύγει τη διαφορά 10. Χρησιμοποίησε το άτοπο: αν δεν υπάρχουν τέτοια ζεύγη, πόσα στοιχεία μπορεί να έχει το \(S\) μέχρι το \(N\);

🟢 2. Αναλλοίωτα

🟢 2. Invariants

Τι μένει σταθερό ενώ όλα αλλάζουν

Πρόβλημα: Σε κύκλο γράφουμε 1,2,…,n. Επιτρέπεται να αντικαταστήσεις δύο γειτονικούς με το άθροισμά τους. Μπορείς να φτάσεις όλους ίσους;

Τι συμβαίνει με το άθροισμα όλων mod 2; Αν ξεκινήσεις με περιττό n, μπορείς να φτάσεις όλους ίσους;

Πρόβλημα: 3 κουτιά με μπίλιες. Επιτρέπεται να διπλασιάσεις ένα κουτί παίρνοντας από τα άλλα δύο. Μπορείς να φτάσεις ίσες μπίλιες σε όλα;

Θεώρησε το άθροισμα mod 3. Τι συμβαίνει όταν διπλασιάζεις;

Πρόβλημα: Σε 5×5 πίνακας όλα 0 εκτός από ένα 1. Επιτρέπεται να αλλάζεις πρόσημο ολόκληρης γραμμής/στήλης. Μπορείς να φτάσεις μόνο 0;

Θεώρησε το άθροισμα κάθε γραμμής ως διάνυσμα στο \(\mathbb F_2^5\). Τι μένει αναλλοίωτο;

🟣 3. Ανάστροφη Σκέψη (Backward Thinking)

🟣 3. Backward Thinking

Ξεκίνα από τον στόχο — όχι από τα δεδομένα

Σε πολλά προβλήματα, τα δεδομένα είναι πολλά, ανοργάνωτα και δεν δείχνουν ξεκάθαρα από πού να ξεκινήσεις.

Η στρατηγική της Ανάστροφης Σκέψης προτείνει να αλλάξεις κατεύθυνση: μην ξεκινήσεις από τα δεδομένα. Ξεκίνα από το τελικό ζητούμενο.

Αναρωτήσου: «Τι θα αρκούσε να ισχύει ώστε το ζητούμενο να είναι αληθές;» Επανάλαβε την ερώτηση, μέχρι να συνδεθείς με όσα δίνονται.

Πώς σκέφτεσαι

  1. Γράψε καθαρά το τελικό ζητούμενο
  2. Ρώτησε: «Τι θα ήταν αρκετό για να ισχύει;»
  3. Μετέτρεψε το ζητούμενο σε απλούστερες συνθήκες
  4. Σύνδεσε τις συνθήκες με τα δεδομένα

Προβλήματα

Πρόβλημα 1. Να αποδείξετε ότι μια δεδομένη ανισότητα ισχύει για όλους τους πραγματικούς αριθμούς. Ξεκινήστε από το ζητούμενο και αναλύστε το σε ισοδύναμες μορφές.

Πρόβλημα 2. Δίνεται τρίγωνο με συγκεκριμένες ιδιότητες. Να αποδείξετε ότι ένα σημείο ικανοποιεί μια γεωμετρική συνθήκη. Ξεκινήστε από το ζητούμενο συμπέρασμα.

Πρόβλημα 3. Να δείξετε ότι μια παράσταση παίρνει ακέραιες τιμές για όλους τους ακέραιους αριθμούς. Ξεκινήστε από την ακέραιη μορφή που θα έπρεπε να έχει.

In many problems, the given information is abundant and does not suggest a clear starting point.

The Backward Thinking strategy reverses direction: do not start from the data. Start from the final goal.

Ask: “What would be sufficient for the conclusion to hold?” Repeat the question until it connects to the given information.

How to Think

  1. State the final goal clearly
  2. Ask what would imply it
  3. Simplify the required conditions
  4. Link them to the given data

Problems

Problem 1. Prove that a given inequality holds for all real numbers. Start from the conclusion and rewrite it into equivalent forms.

Problem 2. A triangle with given properties is provided. Show that a point satisfies a specific geometric condition. Begin from the desired conclusion.

Problem 3. Show that an expression takes integer values for all integers. Start from the form it would need to have.

🟠 4. Αναδιατύπωση (Reformulation)

🟠 4. Reformulation

Πες το ίδιο πρόβλημα με άλλα λόγια

Πολλά μαθηματικά προβλήματα δεν είναι δύσκολα επειδή είναι βαθιά, αλλά επειδή είναι κακοδιατυπωμένα. Η ουσία υπάρχει, αλλά είναι κρυμμένη μέσα σε περιττές λεπτομέρειες, σύμβολα ή αφηρημένη γλώσσα.

Η στρατηγική της Αναδιατύπωσης σε καλεί να μην επιτεθείς άμεσα στο πρόβλημα. Αντίθετα: ξαναπές το πρόβλημα με διαφορετική μορφή, χωρίς να αλλάξεις το νόημά του.

Μια σωστή αναδιατύπωση μπορεί να μετατρέψει ένα «δύσκολο» πρόβλημα σε προφανές.

Πώς σκέφτεσαι

  1. Ρώτησε: «Τι λέει πραγματικά αυτό το πρόβλημα;»
  2. Αφαίρεσε λέξεις ή δεδομένα που δεν επηρεάζουν το ζητούμενο
  3. Αντικατάστησε έννοιες με ισοδύναμες (π.χ. αριθμούς με ιδιότητες)
  4. Ξαναγράψε το πρόβλημα σε απλούστερη μορφή

Προβλήματα

Πρόβλημα 1. Δίνεται ένα πρόβλημα με πολύπλοκη εκφώνηση που περιλαμβάνει πολλές σχέσεις. Να αναδιατυπώσετε το πρόβλημα ώστε να εκφράζεται μόνο με μία βασική ιδιότητα.

Πρόβλημα 2. Ένα γεωμετρικό πρόβλημα περιγράφεται με πολλές γωνίες και ευθείες. Να το αναδιατυπώσετε χρησιμοποιώντας μόνο αποστάσεις.

Πρόβλημα 3. Ένα πρόβλημα θεωρίας αριθμών αναφέρεται σε διαιρετότητες. Να το αναδιατυπώσετε ως πρόβλημα ισοτιμίας modulo.

Many mathematical problems are not difficult because they are deep, but because they are poorly phrased. The core idea exists, but it is hidden behind unnecessary details, symbols, or abstraction.

The Reformulation strategy suggests not attacking the problem directly. Instead: restate the problem without changing its meaning.

A good reformulation can turn a difficult problem into an obvious one.

How to Think

  1. Ask: “What is this problem really saying?”
  2. Remove details that do not affect the goal
  3. Replace concepts with equivalent ones
  4. Rewrite the problem in a simpler form

Problems

Problem 1. A problem is given with a long and complex statement. Reformulate it so that it depends on a single core property.

Problem 2. A geometric problem is described using many angles and constructions. Reformulate it using distances only.

Problem 3. A number theory problem involves divisibility. Reformulate it as a modular arithmetic problem.

🟦 5. Ουσιαστικές Περιπτώσεις (Meaningful Cases)

🟦 5. Meaningful Cases

Διάσπαση σε περιπτώσεις — αλλά μόνο τις σωστές

Η διάσπαση σε περιπτώσεις είναι από τις πιο συχνές στρατηγικές στη μαθηματική σκέψη — και ταυτόχρονα από τις πιο επικίνδυνες. Αν χωρίσεις αυθαίρετα, το πρόβλημα γίνεται γρήγορα χαοτικό.

Η στρατηγική των Ουσιαστικών Περιπτώσεων δεν λέει «χώρισε σε όλες τις δυνατές περιπτώσεις», αλλά: χώρισε μόνο όταν αλλάζει η φύση του προβλήματος.

Οι σωστές περιπτώσεις είναι λίγες, καθαρές και αμοιβαία αποκλειόμενες.

Πώς σκέφτεσαι

  1. Ρώτησε: «Πότε αλλάζει ουσιαστικά το πρόβλημα;»
  2. Εντόπισε μία ή δύο κρίσιμες ιδιότητες
  3. Δημιούργησε περιπτώσεις βασισμένες σε αυτές
  4. Βεβαιώσου ότι οι περιπτώσεις δεν επικαλύπτονται

Προβλήματα

Πρόβλημα 1. Να λυθεί εξίσωση που περιέχει απόλυτες τιμές, χωρίζοντας το πρόβλημα σε περιπτώσεις ανάλογα με το πρόσημο των εκφράσεων.

Πρόβλημα 2. Δίνεται σημείο στο επίπεδο και κύκλος. Να μελετηθεί μια ιδιότητα ανάλογα με το αν το σημείο βρίσκεται εντός, επί ή εκτός του κύκλου.

Πρόβλημα 3. Δίνεται ακέραιος αριθμός \(n\). Να εξεταστεί μια ιδιότητα του \(n\) χωρίζοντας σε περιπτώσεις ανάλογα με το αν είναι άρτιος ή περιττός.

Case analysis is one of the most common strategies in mathematical reasoning — and also one of the most dangerous. If cases are chosen arbitrarily, the problem quickly becomes chaotic.

The Meaningful Cases strategy does not say “split into all possible cases”, but: split only when the nature of the problem changes.

Good cases are few, clean, and mutually exclusive.

How to Think

  1. Ask: “When does the problem truly change?”
  2. Identify one or two critical properties
  3. Build cases based on those properties
  4. Ensure cases do not overlap

Problems

Problem 1. Solve an equation involving absolute values by splitting into cases according to the sign of expressions.

Problem 2. A point and a circle are given. Study a property depending on whether the point lies inside, on, or outside the circle.

Problem 3. An integer \(n\) is given. Study a property of \(n\) by splitting into cases depending on whether it is even or odd.

🟠 6. Συμμετρία – Αντισυμμετρία

🟠 6. Symmetry – Antisymmetry

Όταν το πρόβλημα δεν ξεχωρίζει κανένα στοιχείο

Υπάρχουν προβλήματα στα οποία όλα τα αντικείμενα παίζουν ακριβώς τον ίδιο ρόλο. Κανένα δεν ξεχωρίζει, κανένα δεν φαίνεται «πιο σημαντικό».

Σε τέτοιες περιπτώσεις, κάθε προσπάθεια να διαλέξεις ένα συγκεκριμένο στοιχείο είναι αυθαίρετη και συχνά αποτυγχάνει. Η σωστή κίνηση είναι να σεβαστείς τη συμμετρία του προβλήματος.

Η συμμετρία δεν είναι διακοσμητικό στοιχείο — είναι πληροφορία. Σου λέει τι δεν μπορεί να διαφέρει και ποιες ποσότητες αναγκαστικά πρέπει να είναι ίσες.

Η αντισυμμετρία χρησιμοποιείται όταν υποθέτεις ότι κάτι διαφέρει και οδηγείσαι σε αντίφαση.

Πώς σκέφτεσαι

  1. Εντόπισε ποια αντικείμενα είναι ισοδύναμα
  2. Ρώτησε: «Τι αλλάζει αν τα ανταλλάξω;»
  3. Συμπέρανε ότι ορισμένα μεγέθη πρέπει να είναι ίσα
  4. Αν χρειάζεται, υπόθεσε ανισότητα και αναζήτησε άτοπο

Προβλήματα

Πρόβλημα 1. Δίνονται θετικοί αριθμοί \(a,b,c\) με συμμετρικές συνθήκες. Να εξεταστεί πότε μια συμμετρική έκφραση παίρνει ακραία τιμή.

Πρόβλημα 2. Σε κανονικό πολύγωνο, να αποδειχθεί ότι ορισμένες αποστάσεις ή γωνίες είναι ίσες χωρίς χρήση υπολογισμών.

Πρόβλημα 3. Δίνεται κατανομή αντικειμένων με πλήρη συμμετρία ως προς τις θέσεις. Να δείξετε ότι δεν μπορεί να υπάρξει «προνομιακό» αντικείμενο.

Some problems treat all objects in exactly the same way. No element stands out as special.

In such situations, choosing a specific object is arbitrary and often leads nowhere. The correct move is to respect the symmetry of the problem.

Symmetry is not decoration — it is information. It tells you what cannot differ and which quantities must be equal.

Antisymmetry is used by assuming inequality and reaching a contradiction.

How to Think

  1. Identify which objects are equivalent
  2. Ask: “What changes if I swap them?”
  3. Conclude that certain quantities must be equal
  4. If needed, assume inequality and seek contradiction

Problems

Problem 1. Positive numbers \(a,b,c\) satisfy symmetric conditions. Study when a symmetric expression attains an extreme value.

Problem 2. In a regular polygon, prove that certain distances or angles are equal without performing computations.

Problem 3. An object distribution is completely symmetric. Show that no object can have a privileged role.

🔷 7. Οριακές Τιμές

🔷 7. Boundary Values

Τι συμβαίνει στα άκρα — όχι στη «μέση»

Πολλά μαθηματικά προβλήματα φαίνονται δύσκολα επειδή προσπαθούμε να τα κατανοήσουμε στην πιο γενική τους μορφή.

Η στρατηγική των Οριακών Τιμών προτείνει μια διαφορετική οπτική: μην ξεκινάς από το γενικό. Ξεκίνα από τα άκρα.

Τι συμβαίνει όταν μια μεταβλητή γίνεται πολύ μικρή, πολύ μεγάλη, ή όταν μια κατανομή γίνεται ακραία; Συχνά, εκεί αποκαλύπτεται τι δεν μπορεί να ισχύει γενικά.

Αν μια πρόταση αποτυγχάνει στις οριακές περιπτώσεις, δεν μπορεί να είναι αληθής σε όλες.

Πώς σκέφτεσαι

  1. Εντόπισε ποια μεγέθη μπορούν να μεταβληθούν
  2. Δοκίμασε ακραίες τιμές ή καταστάσεις
  3. Παρατήρησε τι αλλάζει και τι παραμένει
  4. Χρησιμοποίησε το άκρο ως έλεγχο ή άτοπο

Προβλήματα

Πρόβλημα 1. Δίνονται θετικοί πραγματικοί αριθμοί με σταθερό άθροισμα. Να εξεταστεί πότε μια δοσμένη έκφραση παίρνει μέγιστη ή ελάχιστη τιμή.

Πρόβλημα 2. Δίνεται κατανομή αντικειμένων σε κουτιά. Να διερευνηθεί ποια κατανομή οδηγεί σε ακραία τιμή ενός μεγεθούς.

Πρόβλημα 3. Σε οικογένεια γεωμετρικών σχημάτων με σταθερή περίμετρο, να εξεταστεί ποιο σχήμα οδηγεί σε ακραίο εμβαδό.

Many problems seem difficult because we try to understand them in full generality.

The Boundary Values strategy suggests the opposite approach: do not start with the general case. Start with the extremes.

What happens when a variable becomes very small, very large, or when a distribution becomes extreme? Boundaries often reveal what cannot hold in general.

If a statement fails at the boundary, it cannot be universally true.

How to Think

  1. Identify quantities that can vary
  2. Test extreme values or configurations
  3. Observe what changes and what remains
  4. Use the boundary as a test or contradiction

Problems

Problem 1. Positive real numbers have a fixed sum. Investigate when a given expression attains a maximum or minimum.

Problem 2. Objects are distributed among boxes. Study which distribution produces an extreme value of a quantity.

Problem 3. In a family of geometric figures with fixed perimeter, determine which shape leads to an extreme area.

🟠 8. Μέθοδος Παραμέτρου

🟠 8. Parameter Method

Κάνε το πρόβλημα να «κινείται»

Ορισμένα προβλήματα δεν είναι δύσκολα επειδή είναι πολύπλοκα, αλλά επειδή είναι στατικά. Όλα μοιάζουν παγωμένα και δεν αποκαλύπτεται καμία δομή.

Η Μέθοδος Παραμέτρου εισάγει κίνηση. Προσθέτεις μια μεταβλητή — μια παράμετρο — και αφήνεις το πρόβλημα να μεταβληθεί.

Καθώς η παράμετρος αλλάζει, παρατηρείς πότε μια ιδιότητα εμφανίζεται, πότε εξαφανίζεται και πότε αλλάζει χαρακτήρα. Αυτές οι μεταβάσεις είναι συχνά το κλειδί της λύσης.

Πώς σκέφτεσαι

  1. Εισήγαγε μια φυσική παράμετρο
  2. Μελέτησε το πρόβλημα καθώς η παράμετρος μεταβάλλεται
  3. Εντόπισε κρίσιμες τιμές ή κατώφλια
  4. Χρησιμοποίησε τη μετάβαση για συμπέρασμα ή άτοπο

Προβλήματα

Πρόβλημα 1. Δίνεται ανισότητα που εξαρτάται από πραγματική παράμετρο \(t\). Να μελετηθεί για ποιες τιμές του \(t\) ισχύει για όλους τους πραγματικούς αριθμούς.

Πρόβλημα 2. Ένα σημείο κινείται πάνω σε ευθεία. Να εξεταστεί για ποιες θέσεις του σημείου ισχύει μια δοσμένη γεωμετρική ιδιότητα.

Πρόβλημα 3. Θεωρούμε σύνολο με \(n\) στοιχεία, όπου \(n\) μεταβάλλεται. Να μελετηθεί πότε μια κατασκευή παύει να είναι δυνατή.

Some problems are not difficult because they are complicated, but because they are static. Nothing moves, and structure remains hidden.

The Parameter Method introduces motion. You add a variable — a parameter — and let the problem evolve.

As the parameter changes, properties may appear, disappear, or change behavior. These transition points often unlock the solution.

How to Think

  1. Introduce a natural parameter
  2. Study how the problem changes
  3. Identify critical values or thresholds
  4. Use transitions to conclude or contradict

Problems

Problem 1. An inequality depends on a real parameter \(t\). Determine for which values of \(t\) it holds for all real numbers.

Problem 2. A point moves along a line. Investigate for which positions a given geometric property holds.

Problem 3. A construction depends on an integer \(n\). Study for which values of \(n\) the construction is possible.

🟡 9. Άλγεβρα → Γεωμετρία

🟡 9. Algebra → Geometry

Μετάφρασε τους αριθμούς σε σχήμα

Πολλά αλγεβρικά προβλήματα φαίνονται δυσπρόσιτα επειδή περιλαμβάνουν πολλές εξισώσεις, ανισότητες ή συμβολικές εκφράσεις χωρίς σαφή εικόνα.

Η στρατηγική Άλγεβρα → Γεωμετρία προτείνει μια ριζική αλλαγή οπτικής: δώσε γεωμετρικό νόημα στους αριθμούς.

Όταν μια ανισότητα γίνει απόσταση, μια εξίσωση γίνει τόπος σημείων και μια παράσταση γίνει μήκος, το πρόβλημα συχνά απλοποιείται δραματικά. Η γεωμετρία αποκαλύπτει δομή που η άλγεβρα κρύβει.

Πώς σκέφτεσαι

  1. Ερμήνευσε μεταβλητές ως σημεία ή αποστάσεις
  2. Μετέτρεψε εξισώσεις σε γεωμετρικούς τόπους
  3. Χρησιμοποίησε σχήμα, συμμετρία ή ανισότητες αποστάσεων
  4. Μετάφρασε το συμπέρασμα ξανά σε αλγεβρική μορφή

Προβλήματα

Πρόβλημα 1. Δίνεται ανισότητα με απόλυτες τιμές. Να αποδειχθεί χρησιμοποιώντας γεωμετρική ερμηνεία αποστάσεων στον άξονα.

Πρόβλημα 2. Δίνεται εξίσωση με δύο μεταβλητές. Να περιγραφεί γεωμετρικά το σύνολο των λύσεών της και να εξαχθεί συμπέρασμα.

Πρόβλημα 3. Να βρεθεί το ελάχιστο μιας αλγεβρικής παράστασης μεταφράζοντάς την σε πρόβλημα αποστάσεων στο επίπεδο.

Many algebraic problems feel inaccessible because they involve symbolic expressions without a clear picture.

The Algebra → Geometry strategy changes perspective: give geometric meaning to numbers.

When an inequality becomes a distance, an equation becomes a locus, and an expression becomes a length, the problem often simplifies dramatically. Geometry reveals structure hidden by algebra.

How to Think

  1. Interpret variables as points or distances
  2. Translate equations into geometric loci
  3. Use geometry, symmetry, or distance inequalities
  4. Translate conclusions back into algebra

Problems

Problem 1. An inequality with absolute values is given. Prove it using a geometric interpretation on the number line.

Problem 2. An equation in two variables is given. Describe geometrically the set of its solutions and draw conclusions.

Problem 3. Find the minimum of an algebraic expression by translating it into a distance problem in the plane.

🔷 10. Γεωμετρία → Άλγεβρα

🔷 10. Geometry → Algebra

Δώσε εξισώσεις στο σχήμα

Πολλά γεωμετρικά προβλήματα μοιάζουν καθαρά οπτικά: σχήματα, γωνίες, αποστάσεις, κύκλοι. Όμως η καθαρή γεωμετρία δεν οδηγεί πάντα σε λύση.

Η στρατηγική Γεωμετρία → Άλγεβρα προτείνει να κάνεις το αντίστροφο βήμα: μετάφρασε το σχήμα σε εξισώσεις.

Συντεταγμένες, εξισώσεις ευθειών, κύκλων, διανύσματα, αλγεβρικές σχέσεις μετατρέπουν το πρόβλημα σε κάτι που μπορείς να χειριστείς με ακρίβεια.

Πώς σκέφτεσαι

  1. Επέλεξε κατάλληλο σύστημα συντεταγμένων
  2. Μετέφρασε σημεία, αποστάσεις και γωνίες αλγεβρικά
  3. Χρησιμοποίησε εξισώσεις και ανισότητες
  4. Ερμήνευσε το αποτέλεσμα ξανά γεωμετρικά

Προβλήματα

Πρόβλημα 1. Δίνεται τρίγωνο στο επίπεδο. Να αποδειχθεί μια σχέση αποστάσεων χρησιμοποιώντας συντεταγμένες.

Πρόβλημα 2. Δίνονται σημεία με γεωμετρική ιδιότητα. Να μεταφραστεί η συνθήκη σε αλγεβρική εξίσωση και να εξαχθεί συμπέρασμα.

Πρόβλημα 3. Να αποδειχθεί ότι τρία σημεία είναι συνευθειακά χρησιμοποιώντας διανύσματα ή εξισώσεις ευθειών.

Many geometric problems appear purely visual: figures, angles, distances. Yet pure geometry does not always lead to a solution.

The Geometry → Algebra strategy takes the opposite route: translate the figure into equations.

Coordinates, line and circle equations, vectors, and algebraic relations turn geometry into a precise, manageable system.

How to Think

  1. Choose a convenient coordinate system
  2. Translate points, distances, and angles algebraically
  3. Use equations and inequalities
  4. Interpret the result geometrically

Problems

Problem 1. A triangle is given in the plane. Prove a distance relation using coordinates.

Problem 2. Points satisfy a geometric condition. Translate it into an algebraic equation and draw conclusions.

Problem 3. Prove that three points are collinear using vectors or line equations.

🔴 11. Αντιστροφή Ερωτήματος

🔴 11. Reverse Question

Ρώτησε το αντίθετο από αυτό που σου ζητούν

Υπάρχουν προβλήματα που «κολλάνε» όχι επειδή είναι δύσκολα, αλλά επειδή το ερώτημα είναι διατυπωμένο με τρόπο που μπλοκάρει τη σκέψη.

Η στρατηγική της Αντιστροφής Ερωτήματος προτείνει ένα αποφασιστικό βήμα: μην απαντήσεις αυτό που σε ρωτούν. Ρώτησε: «Πότε ΔΕΝ θα ίσχυε αυτό;»

Συχνά, το αντίθετο ερώτημα είναι πολύ πιο εύκολο. Αποκαλύπτει τις αναγκαίες συνθήκες και ξεκαθαρίζει τι πραγματικά παίζει ρόλο.

Πώς σκέφτεσαι

  1. Γράψε καθαρά το αρχικό ζητούμενο
  2. Διατύπωσε το αντίθετο ή την άρνησή του
  3. Μελέτησε πότε αυτή η άρνηση είναι δυνατή
  4. Εντόπισε τις αναγκαίες συνθήκες
  5. Επέστρεψε στο αρχικό ερώτημα

Προβλήματα

Πρόβλημα 1. Να αποδειχθεί ότι μια ιδιότητα ισχύει για όλους τους φυσικούς αριθμούς. Μελετήστε πρώτα πότε δεν θα μπορούσε να ισχύει.

Πρόβλημα 2. Να δείξετε ότι μια γεωμετρική κατασκευή είναι μοναδική. Εξετάστε πότε θα μπορούσαν να υπάρχουν δύο διαφορετικές.

Πρόβλημα 3. Να αποδειχθεί ότι μια εξίσωση έχει ακριβώς μία λύση. Ξεκινήστε με το ερώτημα πότε θα είχε περισσότερες.

Some problems feel impossible not because they are difficult, but because the question itself blocks intuition.

The Reverse Question strategy suggests a bold move: do not answer the question directly. Instead ask: “When would this NOT be true?”

The opposite question is often simpler. It reveals necessary conditions and clarifies what truly matters.

How to Think

  1. State the original question clearly
  2. Formulate its negation
  3. Analyze when the negation can occur
  4. Extract necessary conditions
  5. Return to the original problem

Problems

Problem 1. Prove that a property holds for all natural numbers. First analyze when it would fail.

Problem 2. Show that a geometric construction is unique. Investigate when two different constructions could exist.

Problem 3. Prove that an equation has exactly one solution. Begin by asking when it could have more than one.

🟢 12. Εισαγωγή Βοηθητικού Αντικειμένου

🟢 12. Auxiliary Object

Πρόσθεσε κάτι που δεν σου δίνεται — αλλά χρειάζεται

Υπάρχουν προβλήματα που δεν λύνονται όσο κι αν προσπαθείς να κινηθείς αυστηρά μέσα στα δεδομένα. Όλα φαίνονται σωστά, αλλά κάτι λείπει.

Η στρατηγική της Εισαγωγής Βοηθητικού Αντικειμένου σου επιτρέπει να κάνεις ένα τολμηρό βήμα: να προσθέσεις κάτι που δεν υπάρχει στην εκφώνηση.

Αυτό μπορεί να είναι ένα σημείο, μια ευθεία, ένας κύκλος, μια νέα μεταβλητή, ένα σύνολο ή ακόμη και μια «τεχνητή» κατασκευή. Το βοηθητικό αντικείμενο δεν είναι ο στόχος — είναι το εργαλείο που αποκαλύπτει τη δομή.

Πώς σκέφτεσαι

  1. Αναγνώρισε ότι κάτι λείπει
  2. Ρώτησε: «Τι θα με βοηθούσε να υπήρχε;»
  3. Εισήγαγε το αντικείμενο χωρίς φόβο
  4. Μελέτησε τις νέες σχέσεις που προκύπτουν
  5. Χρησιμοποίησε το συμπέρασμα — όχι το αντικείμενο

Προβλήματα

Πρόβλημα 1. Δίνεται τρίγωνο με ιδιότητα που δεν φαίνεται άμεσα. Να εισαχθεί κατάλληλο βοηθητικό σημείο ώστε να αποκαλυφθεί συμμετρία.

Πρόβλημα 2. Σε συνδυαστικό πρόβλημα καταμέτρησης, προσθέστε ένα «φανταστικό» στοιχείο ώστε να απλοποιηθεί η δομή.

Πρόβλημα 3. Δίνεται αριθμητική σχέση που δεν οδηγεί σε συμπέρασμα. Εισάγετε νέα μεταβλητή ώστε να εμφανιστεί κρυφή διαιρετότητα.

Some problems cannot be solved as long as you stay strictly within the given data. Everything seems correct, yet something is missing.

The Auxiliary Object strategy allows a bold move: add something that is not given.

This could be a point, a line, a circle, a new variable, a set, or even an artificial construction. The auxiliary object is not the goal — it is the tool that reveals hidden structure.

How to Think

  1. Recognize that something is missing
  2. Ask: “What would help if it existed?”
  3. Add the object freely
  4. Study the new relationships
  5. Keep the conclusion — discard the object

Problems

Problem 1. Given a triangle with a hidden property, introduce an auxiliary point to reveal symmetry.

Problem 2. In a counting problem, add a “phantom” element to simplify the structure.

Problem 3. Given a numerical relation, introduce a new variable to expose hidden divisibility.
