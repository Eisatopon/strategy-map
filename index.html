The Extremal Element strategy tells you not to start arbitrarily. Instead, begin with an object in an extreme position: the smallest, the largest, the first, or the last.

An extremal object has fewer possible behaviors than all others. These restrictions often reveal the structure of the problem immediately.

How to Think

  1. Identify an object with an extreme property
  2. Analyze which actions or relations are impossible for it
  3. Use these restrictions to derive a conclusion or contradiction

Problems

Problem 1. In a set of positive integers, the sum of any two distinct elements is different from the sum of any other two. Prove that there exists an element greater than the sum of all the others.

Problem 2. Given n points in the plane, not all collinear, prove that there exists a point whose minimum distance to the other points is unique.

Problem 3. In a finite graph, every vertex has degree at least 2. Prove that there exists a vertex whose removal does not increase the number of connected components.
