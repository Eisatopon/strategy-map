1. Extremal Element

Where to start â€” the most constrained object

In many mathematical problems, the main difficulty is not computation, but deciding where to start.

The Extremal Element strategy is based on a simple principle: start where freedom is minimal.

When to use:
  • "At most" / "at least" statements
  • Minimum or maximum conditions
  • Uniqueness arguments
Thinking steps:
  1. Identify the extremal element
  2. List what it cannot do
  3. See how it affects others
  4. Use for conclusion or contradiction

ğŸ“˜ Basic Level Problem

Given a finite set of positive integers where all pairwise sums are distinct, prove it contains at most one even number.

Hint: What happens if there are two even numbers? Consider the extremal element: the smallest even number. Can you find two pairs with the same sum?

ğŸ“™ Intermediate Level Problem

Given \(n\) points in the plane (not all collinear), prove there exists a point such that, if removed, the minimal distance between remaining points doesn't decrease.

Hint: What is the minimal distance among all points? There's a pair achieving it. What happens if you remove a point not in this pair?

ğŸ“• Advanced Level Problem

In a connected graph with \(n\) vertices, prove there exists a vertex whose removal doesn't increase the number of connected components by more than 1.

Hint: Consider a spanning tree of the graph. Which vertex is the extremal element? (Hint: a leaf has only one edge!)
