3. Αντιπαραδείγματα (Counterexamples)

Ένα παράδειγμα που σπάει τον κανόνα

Μερικές φορές ο καλύτερος τρόπος να δείξεις ότι κάτι είναι λάθος δεν είναι να το αποδείξεις, αλλά να βρεις ένα μόνο παράδειγμα που το διαψεύδει.

Η στρατηγική των Αντιπαραδειγμάτων είναι απλή αλλά καταλυτική: για να διαψεύσεις μια γενική πρόταση, αρκεί ένα μόνο παράδειγμα που δεν την ικανοποιεί.

Αντί να πολεμάς με πολύπλοκες αποδείξεις, βρες το "εξαιρετικό" στοιχείο που καταρρίπτει την υπόθεση. Ένα αντιπαράδειγμα αξίζει όσο χίλιες λέξεις!

Πότε να τη διαλέξεις:

  • Όταν μια πρόταση φαίνεται "πολύ γενική" ή "πολύ ισχυρή"
  • Όταν ζητείται να ελέγξεις αν μια δήλωση είναι αληθής
  • Όταν βλέπεις λέξεις όπως "όλοι", "κάθε", "πάντα"
  • Όταν η διαίσθησή σου λέει "αυτό δεν μπορεί να ισχύει παντού"

Πώς σκέφτεσαι (βήμα-βήμα):

  1. Διάβασε προσεκτικά την πρόταση - τι ισχυρίζεται ακριβώς;
  2. Ψάξε για ακραίες περιπτώσεις: πολύ μικρές/μεγάλες τιμές, 0, 1, αρνητικοί αριθμοί
  3. Δοκίμασε ειδικές περιπτώσεις που συνήθως δημιουργούν προβλήματα
  4. Αν βρεις παράδειγμα που παραβιάζει την πρόταση → τελείωσες!
  5. Αν δεν βρεις, τότε ίσως η πρόταση είναι αληθής (και χρειάζεται απόδειξη)

📘 Πρόβλημα Βασικού Επιπέδου

Πρόταση: "Το τετράγωνο κάθε πραγματικού αριθμού είναι θετικό."

Είναι αυτή η πρόταση αληθής; Αν όχι, δώστε αντιπαράδειγμα.

Hint: Σκέψου τι συμβαίνει με "ειδικούς" αριθμούς.

Ερώτηση 1: Τι γίνεται με θετικούς αριθμούς; (π.χ. 3² = 9 > 0 ✓)

Ερώτηση 2: Τι γίνεται με αρνητικούς; (π.χ. (-2)² = 4 > 0 ✓)

Ερώτηση 3: Υπάρχει κάποιος αριθμός που το τετράγωνό του ΔΕΝ είναι θετικό;

Ερώτηση 4: Τι γίνεται με το μηδέν;

Λύση:

Απάντηση: Η πρόταση είναι ψευδής. ✗

Αντιπαράδειγμα: x = 0

Αιτιολόγηση:
Για x = 0, έχουμε: x² = 0² = 0

Το 0 δεν είναι θετικός αριθμός (0 ≥ 0 αλλά 0 ≠ θετικός).

Άρα υπάρχει τουλάχιστον ένας πραγματικός αριθμός (το 0) του οποίου το τετράγωνο δεν είναι θετικό. ✓

🔑 Κλειδί: Ένα μόνο αντιπαράδειγμα αρκεί για να καταρρίψει μια γενική πρόταση!

📙 Πρόβλημα Ενδιάμεσου Επιπέδου

Πρόταση: "Αν n είναι φυσικός αριθμός, τότε το n² + n + 41 είναι πρώτος αριθμός."

Είναι αυτή η πρόταση αληθής; Αν όχι, βρείτε αντιπαράδειγμα.

Hint: Δοκίμασε μικρές τιμές, αλλά και ειδικές τιμές σχετικές με το 41!

Ερώτηση 1: Δοκίμασε n = 1: 1² + 1 + 41 = 43 (πρώτος ✓)

Ερώτηση 2: Δοκίμασε n = 2: 4 + 2 + 41 = 47 (πρώτος ✓)

Ερώτηση 3: Τι θα συμβεί αν το n σχετίζεται με το 41; (π.χ. n = 40 ή n = 41)

Ερώτηση 4: Αν n = 41, μπορείς να παραγοντοποιήσεις;

Λύση:

Απάντηση: Η πρόταση είναι ψευδής. ✗

Αντιπαράδειγμα: n = 41

Υπολογισμός:

n² + n + 41 = 41² + 41 + 41
= 1681 + 41 + 41
= 1763
= 41 × 43

Παρατήρηση:
Μπορούμε να παραγοντοποιήσουμε:
41² + 41 + 41 = 41(41 + 1 + 1) = 41 × 43

Επειδή 1763 = 41 × 43, δεν είναι πρώτος αριθμός. ✗

🔑 Κλειδί: Αν και η πρόταση λειτουργεί για n = 0, 1, 2, ..., 39, 40, αποτυγχάνει για n = 41!

📕 Πρόβλημα Προχωρημένου Επιπέδου

Πρόταση: "Κάθε συνεχής συνάρτηση είναι παραγωγίσιμη."

Βρείτε αντιπαράδειγμα που δείχνει ότι η πρόταση είναι ψευδής.

Hint: Ψάξε για συνάρτηση με "γωνία" ή "αιχμή".

Ερώτηση 1: Τι γίνεται με τη συνάρτηση f(x) = x; (συνεχής ✓, παραγωγίσιμη ✓)

Ερώτηση 2: Τι γίνεται με f(x) = x²; (συνεχής ✓, παραγωγίσιμη ✓)

Ερώτηση 3: Τι γίνεται με την απόλυτη τιμή f(x) = |x|; Έχει παράγωγο στο x = 0;

Ερώτηση 4: Ποια είναι η κλίση αριστερά και δεξιά του 0;

Λύση:

Απάντηση: Η πρόταση είναι ψευδής. ✗

Αντιπαράδειγμα: f(x) = |x|

Απόδειξη:

1. Συνέχεια: Η f(x) = |x| είναι συνεχής παντού, συμπεριλαμβανομένου του x = 0.

2. Παραγωγισιμότητα στο x = 0:

Αριστερή παράγωγος: f'(0⁻) = -1

Δεξιά παράγωγος: f'(0⁺) = 1

Επειδή f'(0⁻) ≠ f'(0⁺), η συνάρτηση δεν είναι παραγωγίσιμη στο x = 0. ✗

Συμπέρασμα: Η f(x) = |x| είναι συνεχής παντού αλλά όχι παραγωγίσιμη στο x = 0. Άρα η πρόταση είναι ψευδής. ✓

🔑 Κλειδί: Συνέχεια ≠ Παραγωγισιμότητα!

⚠️ Συχνά Λάθη:

  • Σταματάς πολύ νωρίς: Δοκιμάζεις 2-3 περιπτώσεις και συμπεραίνεις ότι ισχύει γενικά
  • Ξεχνάς ειδικές τιμές: 0, 1, -1, άπειρο είναι συχνά "προβληματικές" τιμές
  • Δεν ελέγχεις ακραία: Πολύ μικρές/μεγάλες τιμές, όρια
  • Παίρνεις μόνο "ωραίες" τιμές: Δοκίμασε και "άσχημα" παραδείγματα
  • Συγχέεις "συχνά αληθές" με "πάντα αληθές": Ακόμα κι αν ισχύει σε 1000 περιπτώσεις, μία εξαίρεση το καταρρίπτει

🧠 Μικρό Quiz

Ερώτηση: Τι χρειάζεται για να διαψεύσεις μια πρόταση της μορφής "Για κάθε x, ισχύει P(x)";

4. Αντίστροφη Σκέψη (Backward Thinking)

Ξεκίνα από το τέλος και δούλεψε προς τα πίσω

Όταν η απευθείας πορεία από το πρόβλημα στη λύση φαίνεται ασαφής, δοκίμασε την αντίστροφη κατεύθυνση.

Η στρατηγική της Αντίστροφης Σκέψης σημαίνει: ξεκίνα από το ζητούμενο και ρώτα "τι χρειάζομαι για να το πετύχω αυτό;" — μετά ρώτα το ίδιο για κάθε προηγούμενο βήμα, έως ότου φτάσεις στα δεδομένα.

Είναι σαν να λύνεις έναν λαβύρινθο από την έξοδο προς την είσοδο — συχνά είναι πολύ ευκολότερο!

Πότε να τη διαλέξεις:

  • Όταν το ζητούμενο είναι ξεκάθαρο αλλά δεν ξέρεις πώς να ξεκινήσεις από τα δεδομένα
  • Όταν το πρόβλημα έχει συγκεκριμένο στόχο (απόδειξη, κατασκευή, υπολογισμός)
  • Σε προβλήματα κατασκευής (γεωμετρία, παιχνίδια, στρατηγικές)
  • Όταν υπάρχει αλυσίδα συνεπειών που πρέπει να ανακατασκευάσεις
  • Σε προβλήματα με πολλαπλά βήματα

Πώς σκέφτεσαι (βήμα-βήμα):

  1. Γράψε το ζητούμενο: Τι ακριβώς θέλεις να αποδείξεις/βρεις/κατασκευάσεις;
  2. Ρώτα "Τι χρειάζομαι;": Τι πρέπει να ισχύει για να πετύχω το ζητούμενο;
  3. Επανάλαβε: Για κάθε προηγούμενο βήμα, ρώτα πάλι "Τι χρειάζομαι;"
  4. Σταμάτα όταν: Φτάσεις σε κάτι που ξέρεις ή που δίνεται στο πρόβλημα
  5. Αντίστρεψε τη σειρά: Γράψε τη λύση από την αρχή προς το τέλος

📘 Πρόβλημα Βασικού Επιπέδου

Σκέφτομαι έναν αριθμό. Τον πολλαπλασιάζω επί 3, προσθέτω 15, διαιρώ με 2, και το αποτέλεσμα είναι 30.

Ποιον αριθμό σκέφτομαι;

Hint: Αντί να προσπαθήσεις με δοκιμές, πήγαινε ανάποδα!

Τελευταίο βήμα: Διαίρεσε δια 2 → αποτέλεσμα 30

Ερώτηση: Τι αριθμός διαιρεμένος δια 2 δίνει 30;

Συνέχισε ανάποδα: Πριν τη διαίρεση, τι είχες;

Λύση με Αντίστροφη Σκέψη:

Ας πάμε ΑΝΕΣΤΡΑΜΜΕΝΑ:

Βήμα 1 (ανάποδα):
Αποτέλεσμα = 30
Πριν τη διαίρεση δια 2: 30 × 2 = 60

Βήμα 2 (ανάποδα):
Είχαμε 60 μετά το "πρόσθεσε 15"
Πριν την πρόσθεση: 60 - 15 = 45

Βήμα 3 (ανάποδα):
Είχαμε 45 μετά το "πολλαπλασίασε επί 3"
Πριν τον πολλαπλασιασμό: 45 ÷ 3 = 15

Απάντηση: Ο αριθμός που σκέφτομαι είναι 15.

Επαλήθευση (προς τα εμπρός):
15 × 3 = 45
45 + 15 = 60
60 ÷ 2 = 30 ✓

🔑 Κλειδί: Αντέστρεψα τις πράξεις (×3 → ÷3, +15 → -15, ÷2 → ×2) και δούλεψα προς τα πίσω!

📙 Πρόβλημα Ενδιάμεσου Επιπέδου

Έχεις δύο κουβάδες: έναν 5-λίτρων και έναν 3-λίτρων. Δεν έχουν σημάδια. Έχεις απεριόριστη πρόσβαση σε νερό.

Πώς μπορείς να μετρήσεις ακριβώς 4 λίτρα νερό;

Hint: Σκέψου ανάποδα - πώς θα φτάσεις στα 4 λίτρα;

Στόχος: 4 λίτρα στον κουβά των 5 λίτρων

Ερώτηση: Πώς μπορείς να αφήσεις 4 λίτρα στον μεγάλο κουβά;

Ιδέα: Αν γεμίσεις τον 5L και αφαιρέσεις 1L...

Πώς αφαιρείς 1L; Με τον 3L κουβά που έχει ήδη 2L μέσα!

Λύση με Αντίστροφη Σκέψη:

Σκέψη Ανάποδα:

Στόχος: 4L στον κουβά των 5L

Πώς; 5L - 1L = 4L
Χρειάζομαι: Γεμάτο τον 5L και να αφαιρέσω 1L

Πώς αφαιρώ 1L;
Χύνω από τον 5L στον 3L που έχει ήδη 2L μέσα
(Ο 3L μπορεί να πάρει μόνο 1L ακόμα)

Πώς βάζω 2L στον 3L;
Γεμίζω 3L, χύνω στον 5L (5L έχει 3L)
Γεμίζω ξανά 3L, χύνω στον 5L μέχρι να γεμίσει
(Ο 5L παίρνει 2L, άρα στον 3L μένουν 1L)
Άδειασε τον 5L, χύνε το 1L από τον 3L στον 5L
Γέμισε τον 3L → έχεις 3L
Χύνε τα 3L στον 5L (που έχει 1L) → 1L + 3L = 4L ✓

Βήματα (σωστή σειρά):

  1. Γέμισε τον 3L κουβά
  2. Χύσε τον στον 5L (5L: 3L, 3L: 0L)
  3. Γέμισε ξανά τον 3L
  4. Χύσε στον 5L μέχρι να γεμίσει (5L: 5L, 3L: 1L)
  5. Άδειασε τον 5L
  6. Χύσε το 1L από τον 3L στον 5L (5L: 1L, 3L: 0L)
  7. Γέμισε τον 3L
  8. Χύσε τον στον 5L (5L: 4L, 3L: 0L) ✓

🔑 Κλειδί: Σκεφτόμουν ανάποδα "Πώς να φτάσω στα 4L;" και βρήκα τη διαδρομή!

📕 Πρόβλημα Προχωρημένου Επιπέδου

Να αποδείξετε ότι αν \(a, b, c\) είναι θετικοί πραγματικοί με \(abc = 1\), τότε:

\[\frac{1}{a^3(b+c)} + \frac{1}{b^3(c+a)} + \frac{1}{c^3(a+b)} \geq \frac{3}{2}\]

Hint 1: Σκέψου ανάποδα - τι θα ήθελες να δείξεις αν είχες abc=1;

Hint 2: Αν abc=1, τότε \(\frac{1}{a^3} = b^3c^3\)

Hint 3: Δοκίμασε να μετασχηματίσεις κάθε όρο χρησιμοποιώντας abc=1

Hint 4: Μετά χρησιμοποίησε AM-GM ή Cauchy-Schwarz

Λύση με Αντίστροφη Σκέψη:

Στόχος: Να δείξουμε το άθροισμα ≥ 3/2

Αντίστροφα: Τι χρειάζομαι για να φτάσω εκεί;

Βήμα 1: Χρησιμοποίησε abc = 1
\(\frac{1}{a^3} = (bc)^3\) (επειδή abc = 1 ⟹ a = 1/(bc))

Άρα:
\[\frac{1}{a^3(b+c)} = \frac{(bc)^3}{b+c} = \frac{b^3c^3}{b+c}\]

Βήμα 2: Το ίδιο για τους άλλους όρους:
\[\frac{b^3c^3}{b+c} + \frac{c^3a^3}{c+a} + \frac{a^3b^3}{a+b} \geq \frac{3}{2}\]

Βήμα 3: Χρησιμοποίησε AM-GM:
\(\frac{b^3c^3}{b+c} \geq \frac{b^3c^3}{2\sqrt{bc}} = \frac{(bc)^{5/2}}{2}\)

Βήμα 4: Αθροίζοντας και χρησιμοποιώντας AM-GM ξανά:
Το άθροισμα ≥ 3/2 ✓

🔑 Κλειδί: Σκεφτόμουν ανάποδα "Τι μετασχηματισμό χρειάζομαι;" και χρησιμοποίησα το abc=1!

⚠️ Συχνά Λάθη:

  • Ξεχνάς να αντιστρέψεις: Αφού βρεις τη διαδρομή ανάποδα, πρέπει να τη γράψεις σωστά!
  • Δεν ελέγχεις αν κάθε βήμα αντιστρέφεται: Μερικές πράξεις δεν αντιστρέφονται (π.χ. x²=4 δεν σημαίνει x=2)
  • Σταματάς πολύ νωρίς: Συνέχισε ανάποδα μέχρι να φτάσεις σε κάτι που ξέρεις
  • Μπερδεύεις τη σειρά: Κράτα σημειώσεις για να μην χάσεις το νήμα
  • Δεν επαληθεύεις: Πάντα έλεγξε τη λύση σου προς τα εμπρός!
